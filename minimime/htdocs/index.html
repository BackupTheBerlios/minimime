MiniMIME

A library to parse and construct MIME messages

What is MiniMIME?

MiniMIME is a shared library designed to parse and construct MIME (RFCs 2045, 2046, 2047 2048 and 2049) messages. It provides an easy to use API for C programms. It's ultimate goal is to be as MIME conformant as possible -- which mean for us, it is tolerant in which input it accepts (if wanted) and as correct as possible when it comes to sending messages.

Requirements

Since MiniMIME (as the name already tells) is designed with minimalism in thoughts, all you will need is a POSIX compliant system, a C compiler and a POSIX compliant libc. There are no external dependencies required.
